KINGDOM COME: DELIVERANCE II - THIRD PERSON VIEW TOGGLE
Version 0.2.2

INSTALLATION:
1. Extract all files to: <KC:D 2 installation folder>/Bin/Win64MasterMasterSteamPGO/
2. Launch the game
3. Press F3 (default) to toggle between first and third person views

CONFIGURATION:
Edit KCD2_TPVToggle.ini to customize hotkeys and logging options.

TROUBLESHOOTING:
- Set LogLevel = DEBUG in the INI file
- Check KCD2_TPVToggle.log for details
